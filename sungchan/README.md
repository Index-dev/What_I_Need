# D.Collection. (Development Collection)
<br /><br />

## 스터디 주제
내가 실제로 사용할 페이지를 만들어보기
<br /><br />

## 개발 주제
개발할 때 사용되는 유용한 사이트들을 모아놓은 페이지
<br /><br />

## 개발하게 된 계기

<br /><br />
